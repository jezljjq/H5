函数简介:

当调用ReadDataAddrToBin接口时,并返回bin数据时,插件内部会自动分配一段内存进行返回. 这个接口可以自己设定这个内存,不用插件进行分配. 避免了分配内存的消耗.

函数原型:

long SetReadDataMemoryData(addr,len)

参数定义:

addr长整形数: 缓存地址. 如果是0，表示使用默认的分配内存的方式.

len 整形数: 缓存长度

返回值:

整形数:
0 : 失败.
1 : 成功.

示例:

dm.SetReadDataMemoryData 23432424,4096
value = dm.ReadDataAddrToBin(hwnd,12341234 ,10)
